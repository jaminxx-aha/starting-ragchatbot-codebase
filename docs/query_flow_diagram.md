## 关键配置参数

| 参数 | 值 | 说明 |
|------|-----|------|
| `CHUNK_SIZE` | 800 字符 | 文本分块大小 |
| `CHUNK_OVERLAP` | 100 字符 | 分块重叠区域 |
| `MAX_RESULTS` | 5 | 最大搜索结果数 |
| `MAX_HISTORY` | 2 | 会话历史条数 |
| `EMBEDDING_MODEL` | all-MiniLM-L6-v2 | 嵌入模型 |
| `ANTHROPIC_MODEL` | claude-sonnet-4-20250514 | Claude 模型 |
